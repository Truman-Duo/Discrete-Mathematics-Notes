### 几个常见的不等式及其代数证明

下面几个常见的不等式对于解决一些代数和数论问题有很大帮助，下作介绍并证明。值得注意的是，这里仅作这些不等式的基础形式，使用数学归纳法等方法可以拓展这些结论，由于篇幅限制，这里不做赘述。

**重要不等式**

$a,b \in \R:a^2  + b^2 \geq \pm 2ab$，当且仅当 $a=b$ 时取等

**证明**

因为始终成立 $(a\pm b)^2= a^2 \pm2ab+b^2 \geq 0,a,b \in \R$, 当且仅当$a=b$ 时取等

所以 $a,b \in \R:a^2  + b^2 \geq \pm 2ab$，当且仅当 $a=b$ 时取等

**均值不等式链**（调和平均值 $\leq$ 几何平均值 $\leq$ 算数平均值 $\leq$ 平方和平均值）

$\frac{1}{\frac{\frac{1}{a}+\frac{1}{b}}{2}}[\frac{2ab}{a+b}] \leq \sqrt{ab} \leq \frac{a+b}{2} \leq \sqrt{\frac{a^2 + b^2}{2}}$，$a,b$ 同号，当且仅当 $a=b$ 时取等

**证明**

先证：几何平均值 $\leq$ 算数平均值

因为始终成立: $(\frac{1}{2}a-\frac{1}{2}b)^2 = \frac{1}{4}a^2 - \frac{1}{2}ab + \frac{1}{4}b^2 \geq 0  $，当且仅当 $a=b$ 时取等

所以: $\frac{1}{4}a^2 + \frac{1}{2}ab + \frac{1}{4}b^2 \geq ab$ , 当 $a,b$ 同号时成立: $(\frac{a+b}{2})^2 \geq (\sqrt{ab})^2$，当且仅当 $a=b$ 时取等

所以：$\frac{a+b}{2} \geq \sqrt{ab}$

再证：调和平均值 $\leq$ 几何平均值

令 $\displaystyle x = \frac{1}{a}, y = \frac{1}{b}$ 应用结论 几何平均值 $\leq$ 算数平均值，有：

$\displaystyle \sqrt{xy} \leq \frac{x + y}{2} \Rightarrow   \frac{1}{\sqrt{ab}} \leq \frac{\frac{1}{a} + \frac{1}{b}}{2}$

两边取倒数，反向不等式方向：

$\displaystyle \frac{2}{\frac{1}{a} + \frac{1}{b}} = \frac{2ab}{a + b} \leq \sqrt{ab}$

再证: 算数平均值 $\leq$ 平方和平均值

因为始终成立: $(\frac{1}{2}a-\frac{1}{2}b)^2 = \frac{1}{4}a^2 - \frac{1}{2}ab + \frac{1}{4}b^2 \geq 0  $，当且仅当 $a=b$ 时取等

所以: $\frac{1}{2}a^2 + \frac{1}{2}b^2 \geq \frac{1}{4}a^2 + \frac{1}{2}ab + \frac{1}{4}b^2 $ , 当 $a,b$ 同号时成立: $\sqrt{\frac{a^2+b^2}{2}} \geq \sqrt{(\frac{a+b}{2})^2} = \frac{a+b}{2}$，当且仅当 $a=b$ 时取等

所以：$\sqrt{\frac{a^2+b^2}{2}} \geq \frac{a+b}{2}$

**三角不等式**

$||a|-|b||\leq |a \pm b| \leq |a|+|b|$

**证明**

先证 $|a \pm b| \leq |a| + |b|$

平方两边：

$|a \pm b|^2 = (a \pm b)^2 = a^2 \pm 2ab + b^2 \leq(|a| \pm |b|)^2 = a^2 \pm 2|a||b| + b^2 $（因为 $|a||b| \geq ab, \quad |a|\geq a, \quad |b|\geq b$）

$\Rightarrow |a \pm b| \leq |a| \pm |b|$

再证：$||a| - |b|| \leq |a \pm b|$

由结论 $|a \pm b| \leq |a| + |b|$ 变形得：

$|a| = |(a \pm b) \mp b| \leq |a \pm b| + |b| \Rightarrow |a| - |b| \leq |a \pm b|$

同理得：$|b| - |a| \leq |a \pm b|$

于是：$||a| - |b|| \leq |a \pm b|$

综上：$||a|-|b||\leq |a \pm b| \leq |a|+|b|$

**柯西不等式**

$(a^2+b^2)(c^2+d^2)\geq (ac+bd)^2$

**证明**

考虑恒成立的不等式：

$(ad - bc)^2 \geq 0$

展开得：

$a^2d^2 - 2abcd + b^2c^2 \geq 0$

两边加 $a^2c^2 + b^2d^2$：

$a^2c^2 + a^2d^2 + b^2c^2 + b^2d^2 \geq 2abcd + a^2c^2 + b^2d^2$

即：

$a^2(c^2 + d^2) + b^2(c^2 + d^2) \geq 2abcd + a^2c^2 + b^2d^2$

整理得：

$(a^2 + b^2)(c^2 + d^2) \geq (ac + bd)^2$

等号当且仅当存在实数 $\lambda$ 使得 $\frac{a}{b} = \frac{c}{d} = \lambda$



### 定义 数的几种分类

本章我们将学习一些初等数论的知识。数论是研究整数及其性质的数学分支，在其中我们将接触到整除、公因数、同余、素数分解等基本主题。

从严格的逻辑体系来看，根据第一性原理，我们首先应定义什么是“数”，再定义各种类型的数，最后才能研究它们的性质。在小学、初中和高中阶段，我们已经学习过自然数、有理数、无理数等概念，并对它们作过初步分类。

但如果从现代严谨的数学视角来看，这些定义往往是不完全的，或者在逻辑上不够精确。例如，实数实际上可以通过戴德金分割或柯西列来定义，这些构造涉及实分析甚至集合论的知识。在后续的讲义中我们将简要涉及这方面内容。

事实上，现代数学对“数”本身的抽象定义也是多种多样的，尚无统一标准，但这并不妨碍数论的发展，历史上数论的萌芽远早于集合论和现代数学基础的建立，很多基本结论都产生于“数是自然存在的”这一直觉之上。因此，本章将暂时不从集合论角度构造“数”，而是默认已经具备了对各类数的初步直觉和认识。

下面，我们先对将要在本章中涉及的几类常见“数”做出基础分类，并为后文埋下伏笔。

#### 常见数的基本分类

- **整数**（$\Z$）
   包括正整数、负整数和 $0$.
- **自然数**（$\N$）
   $0,1,2,3,\dots$，
- **分数**
  两个整数之比，通常写作 $\frac{a}{b}$，其中 $b\ne 0$.
- **有理数/ 可比数**（$\Q$）
  所有能写成 $\frac{a}{b}$ 形式的数，$a,b\in\mathbb{Z}, b\ne 0$.
- **无理数**
  非有理数，如 $\sqrt{2}$、$\pi$、$e$ 等.
- **实数**（$\mathbb{R}$）
  所有的有理数和无理数. 实数都可以在一维数轴上找到对应的点.
- **虚数**
  形如 $bi$，其中单位 $i$: $i^2 = -1$, $b\in\mathbb{R}$. 
- **复数**（$\mathbb{C}$）
   形如 $a + bi$ 的数，其中 $a,b\in\mathbb{R}$. 复数都可以在二维数轴上找到对应的点.

### 定义 整除

设 $a,b \in \Z$，同时$b \not= 0$：若 $\exist q \in \Z$ 成立 $ a=b \cdot q$，则此时定义 $b$ 整除 $a$，或称 $a$ 被 $b$ 整除,记为 $b|a$。同时定义$a$ 是 $b$ 倍数，$b$ 是 $a$ 的因数。若不存在 $q$ ，则定义 $b$ 不整除 $a$，或称 $a$ 不被 $b$ 整除, 记为 $b \nmid a$。 

**例 1**

(1) 现有$a,b \in \Z$，证明：

​	(1) $a[这里指代任何非0整数] \mid 0$

​	(2) $1 \mid a[这里指代任何整数]$

​	(3) $a\mid b \Leftrightarrow a\mid |b|$

(2) 现有$a,b,c,d \in \Z$：

​	(1) 若成立$a\mid b, b\mid c$，证明：$a \mid c$

​	(2) 若成立$a\mid b$，证明：$a \mid b \cdot c$

​	(3) 若成立$c\mid a, c\mid b$，证明：$c \mid (a\pm b)$

​	(4) 若成立$c\mid a_i,(i\in\Z_+)$，证明：$c \mid \sum^{n}_{i=1} a_i \cdot d_i$

​	(5) 若成立$c\mid a, c\nmid b$，证明：$c \nmid (a\pm b)$

​	(6) 若成立$a\mid b, c\mid d$，证明：$ac \mid bd$

​	(7) 若成立$a=b+c, d\mid a, d\mid b$，证明：$d \mid c$

**解答**

(1) - (1) 

​	因为始终成立: $0=ak,(k \in\Z\and a\not=0)$

​	所以始终成立: $a[这里指代任何非0整数] \mid 0$

(1) - (2) 

​	因为始终成立: $a=1 \times a$

​	所以始终成立: $1 \mid a[这里指代任何整数]$

(1) - (3) 

​	先证: $a\mid b \Rightarrow a\mid |b|$

​		$a\mid b \Rightarrow b = ak,(k \in \Z)$

​	等式两边同时乘 $-1$：

​		$-b = -ak = a\cdot(-k),(-k \in\Z) \Rightarrow a \mid -b$

​	由已知和上式可知:

​		$(a \mid b) \and (a \mid -b) \Rightarrow a \mid |b|$

​	再证: $a\mid b \Leftarrow a\mid |b|$

​		$a\mid |b| \Rightarrow |b|= ak,(k \in\Z)$

​	当$b\leq 0$ 时:

​		$|b|=-b$, 则此时成立:$-b = ak,(k \in\Z) \Rightarrow a|-b$

​	当$b>0$ 时:

​		$|b|=b$, 则此时成立:$b = ak,(k \in\Z) \Rightarrow a|b$

​	由上式可知:

​		$a \mid |b|\Rightarrow (a \mid b) \and (a \mid -b) $

​	因此始终成立:

​		$a\mid b \Leftrightarrow a\mid |b|$

(2) - (1)

​	由已知：$a \mid b$ 且 $b \mid c$

​	则 $\exist m,n \in \Z:$ 

​		$b = a \cdot m$，$c = b \cdot n$

​	代入得：

​		$c = (a \cdot m) \cdot n = a \cdot (mn)$, $(m,n \in \Z)$

​	因此成立：$a \mid c$

(2) - (2)

​	由已知：$a \mid b$，即存在 $k \in \Z$ 使得：

​		$b = ak$

​		两边同乘 $c$，得：

​		$bc = (ak)c = a(kc)$, $(k,c \in \Z)$

​	因此成立：$a \mid bc$

(2) - (3)

​	由已知：$c \mid a$ 且 $c \mid b$

​	则存在 $m,n \in \Z$ 使得：

​		$a = cm$，$b = cn$

​	则：

​		$a \pm b = cm \pm cn = c(m \pm n)$，其中 $m \pm n \in \Z$

​	因此成立：$c \mid (a \pm b)$

(2) - (4)

​	由已知：对于所有 $i \in \Z_+$，成立 $c \mid a_i$

​	即对每个 $i$，存在 $k_i \in \Z$ 使得：$a_i = c \cdot k_i$

​	则：

​		$\sum_{i=1}^{n} a_i d_i = \sum_{i=1}^{n} (c \cdot k_i) \cdot d_i = c \cdot \sum_{i=1}^{n} k_i d_i$，其中 $\sum k_i d_i \in \Z$

​	因此成立：$c \mid \sum_{i=1}^{n} a_i d_i$

(2) - (5)

​	由已知：$c \mid a$ 且 $c \nmid b$

​	若假设成立：$c \mid (a + b)$，则存在 $m \in \Z$ 使得：$a + b = cm$

​	又由 $c \mid a$，存在 $n \in \Z$ 使得：$a = cn$

​	代入得：

​		$a + b = cn + b = cm \Rightarrow b = c(m - n)$，其中 $m - n \in \Z$

​	故得出：$c \mid b$，与已知 $c \nmid b$ 矛盾

​	同理，若假设 $c \mid (a - b)$ 亦推出 $c \mid b$，也矛盾

​	因此推出：$c \nmid (a \pm b)$

(2) - (6)

​	由已知：$a \mid b$ 且 $c \mid d$

​	则存在 $m,n \in \Z$ 使得：

​		$b = a \cdot m$，$d = c \cdot n$

​	则：

​		$bd = (a \cdot m)(c \cdot n) = (a \cdot c)(m \cdot n)$，其中 $mn \in \Z$

​	因此成立：$ac \mid bd$

(2) - (7)

​	由已知：$a = b + c$，$d \mid a$ 且 $d \mid b$

​	即存在 $m,n \in \Z$ 使得：

​		$a = dm$，$b = dn$

​	代入得：

​		$c = a - b = dm - dn = d(m - n)$，其中 $m - n \in \Z$

​	因此成立：$d \mid c$

### 定义 带余除法

设 $a,b \in \Z$，同时 $b > 0$：$\exist q,r \in \Z$ 始终成立 $ a=b \cdot q +r(0\leq r <b)$。$q$ 称为 $a$ 除以 $b$ 所得到的不完全商，$r$ 称为 $a$ 除以 $b$ 所得到的余数。

### 带余除法的性质

$q,r$ 的取法存在且唯一

**证明**

**存在性**

考虑一维数轴上的一组点序列：$...,-3b,-2b,-b,0,b,2b,3b,...$, 取切段$[kb,(k+1)b),k \in \Z$，显然这组点序列将数轴本身“分切”为了无数份“长度为$b$”的切段。

因此 $a$ 处于其中某一节切段上, 即存在一个整数 $q$ 成立：$qb \leq a < (q+1)b$. 

再令$r = a - bq$, 由此可得：$a = bq +r; \quad  0 \leq r < b$

**唯一性**

假设存在有两种取法，即$a = bq_1 +r_1 = bq_2 +r_2, \quad b > 0,q_{1/2},r_{1/2} \in \Z$

移项有：$b(q_1-q_2) = r_2 - r_1 \Rightarrow |b(q_1-q_2)| = |r_2 - r_1|$

因为 $b >0$, 故：$b|q_1-q_2| = |r_2 - r_1|$

因为$0 \leq r < b \Rightarrow \left\{
\begin{aligned}
  0 \leq r_2 < b \\
  -b < -r_1 \leq 0
\end{aligned}
\right.$, 所以 $-b < r_2 - r_1 < b \Rightarrow |r_2 - r_1| < b$

因此：$|q_1-q_2| < 1, q_{1/2} \in \Z$, 产生矛盾，因此原假设不成立，不存在两种取法。

类似地我们还可以不断做下去得到不存在三种取法，四种取法....

所以$q,r$ 取法唯一得证

(1) 设 $a,b \in \Z$, 且 $b \not = 0$, 证明存在唯一的 $q,r$ 的取法使得$a = bq+r, 0 \leq r < |b|; q,r \in \Z$

​	(1) 任意两个连续整数中必有一个是 $2$ 的倍数；

​	(3) 任意$n$个连续整数中必有一个是 $n$ 的倍数；





你说得**非常好**，这确实是很多教材在讲这种题时**容易忽略的关键点**：我们不能只说“余数只有 $n$ 种”，就想当然认为 $n$ 个连续整数一定包含每一种余数；我们还必须证明：**$n$ 个连续整数在模 $n$ 下的余数不会重复**，即它们在模 $n$ 意义下形成一个**完整的剩余系**（每个余数恰好出现一次）。

下面我们就以证明第（3）问为例，补上这个**核心逻辑**。

------

### 题设重述

**命题：**任意 $n$ 个连续整数中，必有一个是 $n$ 的倍数。

**目标：** 证明对任意整数 $k$，集合 $\{k, k+1, \dots, k+n-1\}$ 中存在某个 $i$ 使得 $k+i \equiv 0 \pmod{n}$。

------

### 补充关键论证（模 $n$ 的余数互不相同）

设任意一个整数 $k$，考虑 $n$ 个连续整数：

$k, k+1, k+2, \dots, k+n-1$

我们令：

$r_i = (k+i) \bmod n, \quad \text{其中 } i = 0, 1, 2, \dots, n-1$

我们要说明：这些 $r_i$ 两两不同，换句话说，余数不会重复，即**每个余数从 $0$ 到 $n-1$ 恰好出现一次**。

------

#### 反证法

假设其中两个数 $k+i$ 和 $k+j$（$0 \le i < j < n$）有相同的余数，即：

$(k+i) \equiv (k+j) \pmod{n} \Rightarrow i \equiv j \pmod{n}$

但由于 $i$ 与 $j$ 都在 $0$ 到 $n-1$ 之间，且 $i \ne j$，所以 $i \not\equiv j \pmod{n}$，这与上面推出的 $i \equiv j \pmod{n}$ 矛盾。

所以假设不成立，说明 $r_0, r_1, \dots, r_{n-1}$ 全部互不相同。

由于它们都是模 $n$ 下的余数，而模 $n$ 的余数一共只有 $n$ 个（即 $0, 1, \dots, n-1$），所以这 $n$ 个余数正好是 $0$ 到 $n-1$ 的一个**排列**。

因此，**余数为 $0$ 必然出现一次**，即存在某个 $i$，使得：

$k + i \equiv 0 \pmod{n} \Rightarrow n \mid (k + i)$

也就是这 $n$ 个连续整数中有一个是 $n$ 的倍数。

✅ 证毕。

------

### 总结归纳（适用于任意 $n$）

- $n$ 个连续整数模 $n$ 下的余数互不相同；
- 模 $n$ 只可能有 $n$ 个不同余数（$0$ 到 $n-1$）；
- 所以这 $n$ 个数的余数必定是 $0,1,\dots,n-1$ 的一个排列；
- 因此必有一个余数为 $0$，即这个数是 $n$ 的倍数。
